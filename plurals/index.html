Regular endings

Normal rules

  • студент → студенты, музей → музеи, автомобиль → автомобили
  • актриса → актрисы, неделя → недели, станция → станции, дверь → двери
  • место → места, здание → здания
  • After г, к, х, ж, ч, ш, щ, use -и: книга → книги, чашка → чашки

Tricky zone

Stress and -а / -я plurals

  • стол → столы́, старик → старики́, игра́ → и́гры, рука́ → ру́ки
  • сестра́ → сёстры, окно́ → о́кна, мо́ре → моря́
  • Some masculine nouns switch to -а / -я: дом → дома́, город → города́, учитель → учителя́

Irregular memory

Core irregulars and frozen forms

  • брат → братья, друг → друзья, сын → сыновья, время → времена́
  • дочь → дочери, мать → матери, ребёнок → де́ти, человек → лю́ди
  • Unchanged in the plural: бюро, метро, такси, кафе, кофе
